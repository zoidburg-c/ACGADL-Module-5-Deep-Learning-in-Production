LlamaIndex: A Brief Introduction

What is LlamaIndex?

Another popular framework for building data-augmented LLM applications, with a focus on data indexing and retrieval.

Core Focus

Specialises in connecting custom data sources to LLMs with sophisticated indexing strategies and query optimisation.

Key Concepts

Documents - Raw data containers

Nodes - Chunked data units

Indices - Data structures for search

Query Engines - End-to-end Q&A

When to Use

Best for structured data and complex queries requiring advanced indexing.
