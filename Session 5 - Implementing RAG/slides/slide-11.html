AWS Bedrock for RAG

What is Bedrock?

AWS managed service providing access to foundation models from multiple providers through a single API.

LangChain Integration

Use the langchain-aws package to seamlessly connect Bedrock models with LangChain's RAG components.

Available Models

Claude (Anthropic) - Advanced reasoning

Titan (Amazon) - General purpose

Llama (Meta) - Open source

Bedrock Embeddings

Amazon Titan Embeddings for creating high-quality vector representations.
