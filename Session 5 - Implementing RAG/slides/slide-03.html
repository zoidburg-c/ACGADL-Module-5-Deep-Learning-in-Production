What is RAG?

RAG = Retrieval-Augmented Generation

The Analogy

RAG is like an open-book exam for AI. Instead of relying purely on memory, the model can look up information before answering.

How It Works

The LLM retrieves relevant documents from a knowledge base, then uses that context to generate accurate, grounded answers.

The LLM can "look up" information before answering, dramatically reducing hallucinations
