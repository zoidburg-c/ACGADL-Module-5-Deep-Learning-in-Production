Document Chunking

Why Chunk?

Documents are too long for embedding models and LLM context windows. Split them into smaller, meaningful pieces.

How It Works

Split text by character count with overlap. The overlap ensures context is not lost at chunk boundaries.

Key Parameters

Chunk Size

500 - 1000 characters per chunk. Balances context with precision.

Overlap

50 - 200 characters. Prevents losing context at boundaries.
