# Module 5 — Deep Learning in Production: Course Materials Design

**Date:** 2026-02-25
**Audience:** Beginners
**Cloud Provider:** AWS
**Assessment:** Short-answer questions (no MCQ) + practical component

## Structure

6 Jupyter notebooks (Sessions 1–5 mixed lecture+hands-on) + 1 assessment Markdown file.

## Sessions

| Session | Title | Duration | Theory / Hands-on |
|---------|-------|----------|-------------------|
| 1 | Introduction to Production Pipelines | 1 hr | 20m / 40m |
| 2 | Model Optimisation Techniques | 1 hr | 20m / 40m |
| 3 | Deploying Models to the Cloud (AWS) | 4 hrs | 60m / 180m |
| 4 | Deploying Models On-premise | 4 hrs | 60m / 180m |
| 5 | Implementing RAG | 6 hrs | 120m / 240m |
| 6 | Review and Assessment | 1 hr | — |

## Key Decisions

- **Format:** Jupyter notebooks with mixed markdown (theory) and code cells (exercises)
- **Session 3:** AWS-focused (SageMaker, ECR, CloudWatch)
- **Session 4:** Both Linux/Nvidia and Apple M-chip/MLX covered
- **Session 5:** LangChain primary, LlamaIndex brief intro
- **Assessment Part B:** Deploy a pre-trained model to AWS SageMaker endpoint
